#XGBOOST
Por medio de XGBoost se busca predecir el puntaje de fatiga mental de una persona.
Las variables independientes que son utilizadas (x):
	age	
	gender	
	occupation	
	daily_screen_time_hours	
	phone_usage_before_sleep_minutes	
	sleep_duration_hours	
	sleep_quality_score	
	stress_level	
	caffeine_intake_cups	
	physical_activity_minutes	
	notifications_received_per_day

La variable dependiente (y):
	mental_fatigue_score
Esta variable se encuentra en el rango entre 0 y 10.

Se busca encontrar la variable con correlación mas fuerte.
Se busca responder a la pregunta de que característica es la que mas afectación tiene en la fatiga mental de una persona.
Se busca entrenar un modelo que sirva para predecir el puntaje de fatiga mental que una persona podría tener basado en las entradas que se asignen.

el dataset utilziado:
https://www.kaggle.com/datasets/jayjoshi37/sleep-screen-time-and-stress-analysis
